# yourman

Make your manual manually.

## how to use

```
$ go get github.com/tkrkt/yman
$ yman --help
```

### add

```
$ yman add [-t tag] [command]
```

Manuals is saved in `~/.yman`

### show

```
$ yman [command]
```

or

```
$ yman show [command]
```

### list

```
$ yman list [-t tag] [command] [--interactive]
```
